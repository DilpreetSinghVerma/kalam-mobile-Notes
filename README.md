**Kalam Notes Mobile** is the official mobile companion to the Kalam Notes desktop application. Access your notes, write on the go, and sync seamlessly using your Google account.

## ✨ Features

*   🌍 **Tri-Language Engine:** Seamlessly toggle between Punjabi, Hindi, and English. 
*   ☁️ **Cloud Sync with Google:** Sign in natively with Google to instantly sync your notes with the PC application.
*   📱 **Responsive Mobile Design:** A stunning UI optimized for touch, featuring responsive grids and list views for your notes.
*   🗑️ **Smart Trash:** Accidentally deleted a note? Recover it from the recently deleted bin before it auto-purges after 30 days.
*   🛡️ **Offline Support:** Write offline! Your notes save locally and sync the moment you reconnect.

## 🚀 Getting Started

### Installation (For Users)
The easiest way to get the app is to navigate to the **[Releases](../../releases)** tab on this repository and download the latest `.apk` file directly to your Android device.

### Development Setup (For Developers)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/DilpreetSinghVerma/kalam-mobile-Notes.git
   cd kalam-mobile-Notes
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run in development mode:**
   ```bash
   npx expo start
   ```

4. **Build APK (Requires EAS CLI):**
   ```bash
   eas build -p android --profile preview
   ```
